Got something to say about ELSA?

  1. Open populate_blog.py.
  2. If you are a new user, add your user to the Atmos Student Users list. Example: Some_New_User = User.objects.get(username='Some_New_User')
  3. Categories currently are not implemented so skip that section.
  4. Copy and paste the commented out post template to the bottom of the list. Uncomment it and fill in the blanks. Ensure you stay within the char limits.
  5. Save populate_blog.py
  6. Run python populate_blog.py in your terminal. The script will let you know how many blog posts were added.

Editing a Post?

  1. Head over to the post section of the admin page and delete your post. Don't worry, this post should be saved in populate_blog.py
  2. Open populate_blog.py.
  3. Edit the post you were wanting to edit
  4. Save populate_blog.py
  5. Run python populate_blog.py in your terminal. The script will let you know if your edited post was re-added.
